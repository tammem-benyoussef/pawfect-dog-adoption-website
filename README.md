# 🐾 Pawfect Home — Dog Adoption Website

A dog adoption website built as a university project using HTML, CSS, JavaScript, PHP, and MySQL.

## ✨ Overview

Pawfect Home helps users browse adoptable dogs, submit adoption applications, and send quick enquiries. It also includes admin pages for reviewing stored applications and enquiries.

## 📄 Pages

| Page | Description |
|---|---|
| `index.html` | Homepage with hero section, featured dogs, adoption process, and testimonials |
| `dogs.html` | Browse all available dogs with search and favorite heart buttons |
| `adopt.html` | Adoption application form with validation and submission feedback |
| `about.html` | Shelter story, mission, team, stats, and gallery |
| `admin_submissions.php` | Admin view for adoption applications |
| `admin_enquiries.php` | Admin view for quick enquiries |

## 🛠️ Built With

- **HTML5** — page structure and content
- **CSS3** — styling, layout, animations, and responsive design
- **JavaScript** — interactive search, heart buttons, form feedback, and enquiry prompts
- **PHP** — backend form processing and enquiry storage
- **MySQL** — database storage for applications and enquiries
- **WAMP or XAMPP** — local development environment for PHP and MySQL

## ✨ Features

- Responsive layout for desktop and mobile
- Featured dog cards with tags and favorite heart buttons
- Search bar for filtering dogs by name or breed
- Adoption application form with client-side validation
- PHP form submission handling
- Quick enquiry flow with JSON submission
- Admin pages to review stored applications and enquiries
- Clean, shelter-inspired visual design
- Smooth scrolling, hover effects, and animated sections
