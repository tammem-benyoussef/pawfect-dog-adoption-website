# 🐾 Pawfect Home — Dog Adoption Website

A static dog adoption website built with HTML and CSS as a university project.

🔗 **Live Preview:** [Click here to visit the website](https://tammem-benyoussef.github.io/pawfect-dog-adoption-website/)

---

## 📄 Pages

| Page | Description |
|---|---|
| `index.html` | Homepage with hero section, featured dogs, and testimonials |
| `dogs.html` | Browse all available dogs with filter tabs |
| `adopt.html` | Adoption application form and requirements |
| `about.html` | About the shelter, team, mission, and gallery |

---

## 🛠️ Built With

- **HTML5** — semantic structure and content
- **CSS3** — styling, layout, animations, and responsive design
- **JavaScript** — heart buttons and form feedback
- **Google Fonts** — Fraunces (headings) and DM Sans (body text)

---

## ✨ Features

- CSS Grid and Flexbox layouts
- Smooth keyframe animations (fade in, floating card, scrolling ticker)
- Interactive dog cards with favourite heart button
- Adoption application form with submission feedback
- Sticky navbar with scroll effect on homepage
- Photo gallery with asymmetric grid layout

## 🚀 How to Run

No installation needed. Just:

1. Clone or download the repository
2. Open `index.html` in any browser

> ⚠️ Requires an internet connection to load Google Fonts.
